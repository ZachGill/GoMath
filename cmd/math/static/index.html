GoMath

GoMath is an API which answers simple math problems

Endpoints
Add

Add two input numbers

REQUEST

GET

/add{?input,input}
Parameters
input number 1

Input number to add

RESPONSE

Headers
Content-Type application/json
X-Transaction-ID "Problem ID"
200 application/json
BODY SCHEMA 
                        {
    "problem_id": "bdmb8cboo3smi6jgiai0",
    "operation": "addition",
    "input_1": 1,
    "input_2": 1,
    "answer": 2
}

Problem

Retrieve result of a specific problem

REQUEST

GET

/problems/{problem_id}
Parameters
problem_id string

Problem ID given in the response header when sending a math problem

RESPONSE

Headers
Content-Type application/json
200 application/json
BODY SCHEMA 
                        {
    "problem_id": "bdmb8cboo3smi6jgiai0",
    "operation": "addition",
    "input_1": 1,
    "input_2": 1,
    "answer": 2
}
